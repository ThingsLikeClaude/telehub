# Git Conventions

## Branch Naming
- feature/feature-name
- bugfix/issue-name
- refactor/target

## Commit Messages
- feat: new feature
- fix: bug fix
- refactor: refactoring
- test: add/modify tests
- docs: documentation changes

## PR Rules
- Title under 70 characters
- Body includes change summary + test plan
- No force push (especially main/master)
- No hook bypass (--no-verify)
