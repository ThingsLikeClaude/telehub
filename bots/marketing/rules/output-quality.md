# Output Quality Standards

## Visual Assets
- Color: Follow brand color system, 3-color max main palette
- Typography: Readability first, clear heading/body hierarchy
- Layout: Generous whitespace, appropriate information density
- Charts/graphs: Axis labels, units, and sources required

## Web Pages
- Mobile responsive required
- Load under 3 seconds (minimize external dependencies)
- CTA buttons clear and prominent
- Basic accessibility (contrast, alt text)

## Documents (PDF/DOCX)
- Include cover page, table of contents, page numbers
- Consistent heading hierarchy
- Captions on all images/charts

## General
- Proofread spelling/grammar
- Filename: `YYYYMMDD_projectname_type.ext`
- Final deliverables must always be sent as files
