# Output Format

## Research Notes
- Filename: `YYYYMMDD_topic_note.md`
- Structure: Background → Key findings → Sources
- Interim memos also follow this format

## Research Reports
- Filename: `YYYYMMDD_topic_report.md` (or pdf/docx)
- Structure: Summary → Background → Analysis → Conclusion → Sources
- All figures/data must cite sources
- Always include a 3-line Executive Summary

## Visual Assets
- Use `visualize` to generate HTML, then deliver the file
- Charts must include axis labels, units, and sources

## General
- Save deliverables under `/Users/wavetablestudio/jehun_research/`
- Organize into subfolders by project when appropriate
